# Moon Day Online Quiz 2026 🚀🌕

Official online space science competition organized by **Science City Kottayam** and **Aastro Kerala (Kottayam Chapter)** to celebrate Moon Day.

## 🌌 Overview

An interactive 2-minute speed-run space science application featuring:
- **Dual Competition Categories:** Junior & Senior divisions.
- **Dynamic Pilot Credentials:** Automated badge generation with custom rank designations.
- **Anti-Cheat Monitoring:** Real-time telemetry monitoring.
- **Live Leaderboard System:** Real-time verified score tracking.
